NovaSite — 配布物
====================
1) 収録ファイル
- index.html（ホーム）
- create.html（ウェブサイト作成フォーム）
- privacy.html（プライバシーポリシー）
- terms.html（利用規約）
- contact.html（お問い合わせフォーム）
- styles.css / script.js
- assets/logo.svg

2) 使い方
- ローカルで開くだけで動作します（静的サイト）。
- お問い合わせフォームを使う場合は、script.js の
  FORMSPREE_CONTACT_ENDPOINT にあなたの Formspree のURLを設定してください。

3) アニメーション
- JavaScript（IntersectionObserver, タイプライター効果）と CSS（グロー・フェードイン）で演出しています。

4) カスタマイズ
- 料金や文言は index.html 内「料金プラン」ブロックを書き換えてください。
- 作成フォームの項目や繰り返し要素は create.html と script.js のテンプレートで変更可能です。
